# Holdout / test texts — not used for training

This folder contains Conan Doyle Sherlock Holmes stories reserved for **evaluation only**. They are excluded from pair generation and training.

- **The_Field_Bazaar.txt** — "The Field Bazaar" (1896)
- **How_Watson_Learned_the_Trick.txt** — "How Watson Learned the Trick" (1924)

Training uses only `data/raw/`. Use these files for holdout evaluation or test-set metrics.
